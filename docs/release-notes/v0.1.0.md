# CIPHER VAULT v0.1.0

初回のデスクトップ版リリースです。
`CIPHER VAULT` を Windows 向けアプリとして配布できるようにし、通常利用向けの `setup` 版と、インストール不要の `portable` 版を公開します。

## 追加

- Electron ベースのデスクトップアプリを追加
- ローカル保存の SQLite 保管庫を実装
- マスターキーによる保管庫の初期設定、解錠、ロックを実装
- サービス名、ログイン ID、パスワード、URL、タグ、メモの管理を実装
- パスワード生成とクリップボードコピーを実装
- 一定時間無操作時の自動ロックを実装
- Windows 向け `setup` / `portable` 配布に対応

## 配布物

- `CIPHER VAULT-0.1.0-x64-setup.exe`
  通常利用向けのインストーラー版です。
- `CIPHER VAULT-0.1.0-x64-portable.exe`
  インストール不要で試しやすい実行版です。

## 注意

- データ保存先は Electron の `userData` 配下です
- `portable` 版でも exe と同じフォルダに DB を保存する方式ではありません
- 初回公開のため、既存バージョンからのアップデートではありません
